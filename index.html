Protein diagrams with pfam domains

Here is a demo of a free to use JavaScript module written by Kristian Gray (GitHub user: KrisGray), a developer working for the HGNC, that will produce interactive SVG images that represent a UniProt protein with Pfam domains in an easy to understand 2D format. The code repository can be found in our HGNC GitHub repository and is released with an open source MIT licence.

Although the code is free and release with a permissive license with very little restrictions, we would appreciate it if you could reference us in some way, however this is not compulsary. The latest reference to our project can be found at the bottom of the page.
